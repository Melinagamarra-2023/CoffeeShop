☕ Coffee Shop - Proyecto Personal
Este es un proyecto de desarrollo web creado desde cero como parte de mi portfolio personal. Se trata de una landing page moderna para una cafetería, diseñada para ofrecer una experiencia visual atractiva y funcional tanto en computadoras como en dispositivos móviles.  

🚀 Características del Proyecto
Proyecto 100% Personal: Diseño y desarrollo íntegramente realizado por mí para demostrar habilidades técnicas en el área Front-End.  

Diseño Responsivo: La interfaz se adapta automáticamente a teléfonos y monitores de escritorio para garantizar una navegación cómoda en cualquier lugar.  

Navegación Interactiva: Incluye un menú lateral (hamburguesa) para móviles y una barra de navegación superior fija que aparece al hacer scroll.  

Experiencia Inmersiva: Uso de videos de alta calidad integrados como fondo de secciones clave para captar la atención del usuario.  

🛠️ Tecnologías Utilizadas
Para este desarrollo decidí utilizar tecnologías estándar de la industria, asegurando un código limpio y eficiente:  

HTML5: Estructura semántica del contenido.  

CSS3: Estilos avanzados, animaciones, Flexbox y CSS Grid para el diseño responsivo.  

JavaScript (Vanilla): Lógica para el manejo del menú, efectos de scroll e interactividad del sitio.
